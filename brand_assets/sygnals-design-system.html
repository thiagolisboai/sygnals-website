06 · Uso e restrições

Fonte

Logo usa DM Sans 700. UI usa Inter Variable. Identidades tipográficas separadas — marca e produto são linguagens distintas.

Next.js

Carregar via next/font/google, weight: '700'. SVG inline no componente — evita FOUT e garante alinhamento consistente.

Cores

Dark: #00d4f5. Light: #0094b3. Nunca usar #00d4f5 sobre branco — contraste insuficiente.

Tamanho mínimo

Logo completo: 172px wide. Abaixo disso, mark isolado. Mark mínimo: 20px. Favicon: 16px com dot proporcional.

Alinhamento

Sempre SVG com dominant-baseline="middle". Nunca HTML+CSS para o logo — alinhamento entre mark e wordmark não é garantido fora do SVG.

Não fazer

Não rotacionar. Não distorcer proporções. Não trocar DM Sans. Não usar stroke-linecap="round" — destrói o caráter angular do mark.
